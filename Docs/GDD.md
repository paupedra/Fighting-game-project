# Game Design Document

## Concepto

+ Título: Pixel Fighters

+ Estudio/Diseñadores: LEG Studios  

+ Género: Fighter 2D, Plataformas, Arcade.

+ Plataforma: PC, Switch

+ Versión: 000

+ Sinopsis de Jugabilidad y Contenido: En un mundo de fantasía se unen personajes de diferentes universos para luchar hasta la muerte.

+ Categoría: Fighter 2D, Parecido a Super Smash Bros, Brawlhalla, Rivals of Aether, etc.

+ Mecánica: El jugador podrá elegir entre varios personajes y tendrá diferentes técnicas para atacar en todas las direcciones, esquivar y doble salto.

+ Tecnología: Unity, C#, Photoshop, Aseprite, FLStudio.

+ Público: Va dirigido a todos los públicos, pero enfocado especialmente en adolescentes.
